# Welcome to my budget tracker app

**Web App Features**
✅ Authentication - Signup/signin with email verification and protected routes
✅ Dashboard - Total balance, monthly income/expenses, budget progress bars, pie chart
✅ Bank Management - Full CRUD for bank accounts with balance tracking
✅ Budget Categories - Percentage-based allocation (Needs/Wants/Investments/Emergency)
✅ Purposes - Organize expenses by purpose linked to categories
✅ Transactions - Record income/expenses with automatic balance updates
✅ Records History - Filterable table by month, bank, and category
✅ Settings - Profile management

**Installation instruction**
(Use your preferred IDE)
Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
